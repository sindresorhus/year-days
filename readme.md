# year-days

> Get the number of days in a year


## Install

```
$ npm install year-days
```


## Usage

```js
const yearDays = require('year-days');

yearDays(2014);
//=> 365

yearDays(2016);
//=> 366
```


## API

### yearDays([year])

#### year

Type: `number`<br>
Default: Current year

### yearDays([date])

#### date

Type: `Date`<br>
Default: `new Date()`
