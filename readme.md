# year-days

> Get the number of days in a year

## Install

```
$ npm install year-days
```

## Usage

```js
import yearDays from 'year-days';

yearDays(2014);
//=> 365

yearDays(2016);
//=> 366
```

## API

### yearDays(year?)

#### year

Type: `number`\
Default: Current year

### yearDays(date?)

#### date

Type: `Date`\
Default: `new Date()`
